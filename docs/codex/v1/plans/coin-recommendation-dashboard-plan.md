# 推荐币子系统调度与可视化计划

## 目标

实现一个本地常驻服务，支持每小时自动执行推荐分析，并提供可视化页面。

## 实施步骤

1. 抽取推荐分析核心执行函数，兼容 CLI 与服务复用。
2. 新增服务端脚本，实现调度器、运行状态管理与 HTTP API。
3. 新增前端页面和静态资源，接入 API 展示结果。
4. 运行本地验证，至少验证服务启动、首轮执行、页面可访问。

## 验证方式

- `python -m py_compile binance_day_contract_realtime_v5.py coin_recommendation_dashboard.py`
- 启动服务后访问首页与 API。

## 风险

- 首轮分析耗时较长，因此服务需要异步执行，不阻塞页面打开。
